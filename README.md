#  Lettycia Vitória | Portfolio

Bem-vindo ao repositório do meu portfólio! Este projeto foi desenvolvido para centralizar minha jornada acadêmica e profissional, meus projetos de desenvolvimento e minhas competências técnicas em um único lugar interativo e responsivo.

### Principais Funcionalidades:
- **Responsividade:** Totalmente adaptado para dispositivos móveis, tablets e desktop.
- **Modo Dark/Light:** Alternância de temas persistente.
- **Jogo da Forca:** Jogo da Forca com palavras relacionadas a programação.

##  Tecnologias Utilizadas

- **Framework:** [Next.js 14+](https://nextjs.org/)
- **Estilização:** [Tailwind CSS](https://tailwindcss.com/)
- **Ícones:** [Lucide React](https://lucide.dev/)
- **Linguagem:** TypeScript / JavaScript
- **Animações:** CSS Keyframes & Tailwind Arbitrary Variants
